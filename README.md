# CampusShop - Tienda Universitaria

Este proyecto es una página web creada para que los estudiantes puedan comprar artículos de la universidad de forma sencilla.

## Estructura del proyecto

El sitio está organizado en diferentes archivos para que cada uno cumpla una función:

### Páginas (Archivos HTML)
* **index.html**: Es la portada de la tienda donde puedes buscar productos.
* **catalogo.html**: Muestra la lista de todos los productos con filtros.
* **producto.html**: Muestra los detalles de un producto específico (tallas, precio).
* **carrito.html**: Es la lista de productos que el usuario quiere comprar.
* **vacio.html**: Se muestra cuando el carrito no tiene nada adentro.
* **checkout.html**: El formulario para poner la dirección y pagar.
* **perfil.html**: Información del usuario.
* **historial.html**: Lista de compras que ya se hicieron.

### Diseño (Archivos CSS)
* **base.css**: Define los colores y el tipo de letra.
* **layout.css**: Organiza dónde van el menú y el pie de página.
* **components.css**: Define cómo se ven los botones y las tarjetas.
* **responsive.css**: Hace que la página se vea bien en celulares.

---

## Cómo navegar por las vistas

Para usar la tienda se sigue este orden:
1. Se entra al **Inicio** para ver qué hay de nuevo.
2. Se va a la **Tienda** para filtrar y buscar productos.
3. Se entra a un **Producto** para ver sus detalles y agregarlo al carrito.
4. En el **Carrito** se revisa el pedido y se da clic en comprar.
5. En el **Checkout** se ponen los datos de entrega y el pago.
6. En el **Perfil** se puede revisar el **Historial** para ver pedidos pasados.
